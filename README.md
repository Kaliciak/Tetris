# Tetris
Project for Java classes.
# Controls
up, x - rotate clockwise\
z - rotate counterclockwise\
left - move left\
right - move right\
down - soft drop\
spacebar - hard drop\
c - hold\
p - pause\
r - restart (if game is over)\
m - mute
#  How to run project
You can compile project with `build.sh` and then run it with `run.sh`.
